# Skypro Music

Музыкальный сервис Skypro.Music — первое задание курса.

## Запуск

```bash
npm install
npm run dev
```

Приложение: [http://localhost:3000](http://localhost:3000)

## Страницы

- `/` — главная (плейлист треков)
- `/signin` — вход
- `/signup` — регистрация

## Структура

- `src/app` — страницы Next.js App Router
- `src/components` — компоненты с CSS Modules
- `public/img` — изображения и иконки
